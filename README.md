# Guideline
### Required installation
- node 
- npm 

(https://nodejs.org/en/download)

### Setup project and dependency
```bash 
git clone https://github.com/sunsonsu/bubble-sort
```

```bash 
cd bubble-sort
```

```bash 
npm install
```

### How to Execute code 
```bash 
npx tsx src/merge.ts
```

### How to Run Unit Tests
```bash 
npm test
```
